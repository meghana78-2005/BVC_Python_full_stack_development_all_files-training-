JavaScript:(converting static pages to dynamic pages)

- used to develop client side as well as server side.
- JS is a single threaded and synchronous programming language which is used for converting static pages to dynamic pages.
- synchronous -> line by line execution.
- asynchronous -> executes all lines of code at a time
- js is by default synchronous but we can achieve asynchronous through callbacks,promises
- single thread -> single task execute at a single time

synchrnous code - no delay
asynchronous code - delay
eg:
	const a = 0
	const b = 1
	console.log(a+b)  # which is a synchronous code
